# Dashlogistica
Projeto: Análise de Atrasos Logísticos — LogiX Brasil

Desenvolvi este projeto com o objetivo de simular um cenário real de negócio, onde uma empresa enfrenta dificuldades no controle de prazos de entrega e precisa identificar os principais gargalos logísticos.

Utilizando Power BI, construí um dashboard interativo com foco em análise de desempenho operacional, permitindo visualizar:
✔️ Percentual de entregas atrasadas
✔️ Tempo médio de entrega
✔️ Desempenho por transportadora
✔️ Regiões e estados com maior índice de atraso

💡 Principais insights:

• Mais de 50% das entregas analisadas ocorreram com atraso
• Os atrasos não são aleatórios, apresentando concentração em regiões específicas
• Diferença significativa de desempenho entre transportadoras
• Identificação de estados com comportamento crítico

🧠 Análise estratégica:

A análise revelou um padrão consistente de ineficiência logística, indicando que os atrasos estão relacionados a fatores operacionais e geográficos, e não a eventos pontuais.

Esse tipo de comportamento sugere falhas estruturais no processo logístico, como rotas mal otimizadas e variação na qualidade das transportadoras.

🚀 Recomendações:

• Reavaliar o desempenho das transportadoras com maior índice de atraso, considerando substituição ou renegociação de contratos com base em SLA.
• Otimizar rotas logísticas nas regiões críticas
• Implementar monitoramento contínuo de indicadores (KPIs)
• Padronizar prazos de entrega para maior previsibilidade operacional

📌 Conclusão:

Os dados demonstram que é possível transformar informações operacionais em decisões estratégicas, contribuindo para a redução de custos e melhoria da experiência do cliente.
